# Security

## Reporting a vulnerability

Please **do not** open a public issue for security-sensitive reports.

**Preferred:** use [GitHub Security Advisories](https://github.com/JeelVankhede/backend-ai-starter-recipes/security/advisories/new) to disclose privately.

**Alternative:** email **jeelvankhede@gmail.com** with a clear subject (e.g. `[security] backend-ai-starter-recipes`).

Include steps to reproduce, affected versions if known, and impact. You should receive an initial response within a reasonable time; critical issues are prioritized.

## Scope

This project is a **CLI that generates files** in a directory you choose. It does not run remote services as part of normal use. Reports about template content or dependency advisories are welcome; use the same channels above.
